
# 🌿 Растения — учёт продаж

Локальное веб-приложение для учёта продаж растений, остатков и расчёта зарплат. Написано на Go без сторонних зависимостей. Данные хранятся в одном JSON-файле на компьютере.

---

## Возможности

- **Продажи** — добавление продаж с автоматическим списанием остатков
- **Расходы** — учёт по категориям (закупка, аренда, транспорт и др.)
- **Остатки** — каталог растений с цветовыми статусами наличия
- **Импорт CSV** — загрузка прайс-листа из Google Sheets одним кликом
- **Зарплаты** — автоматический расчёт по % от чистой прибыли
- **Дашборд** — выручка, расходы, прибыль, топ-5 растений, разбивка по каналам продаж

---

## Стек

- **Сервер:** Go 1.21+, стандартная библиотека net/http
- **База данных:** JSON-файл (data.json)
- **Фронтенд:** Vanilla HTML/CSS/JS, без фреймворков
- **Зависимости:** Ноль — только стандартная библиотека Go

---

## Быстрый старт

### Требования
- Go 1.21+
- Windows / macOS / Linux

### Запуск

git clone https://github.com/cacarerru-prog/plan-accounting.git
cd plan-accounting
go run main.go

Открой браузер: http://localhost:8080

## API

| Метод | Путь | Описание |
|-------|------|----------|
| GET | /api/plants | Список растений |
| POST | /api/plants | Добавить растение |
| GET | /api/sales | История продаж |
| POST | /api/sales | Добавить продажу |
| GET | /api/expenses | Список расходов |
| POST | /api/expenses | Добавить расход |
| GET | /api/stats | Статистика и топ-5 |
| POST | /api/import/csv | Импорт из CSV |
